## :rocket: Como executar 
* Clone o repositório:
```` git clone https://github.com/MarceloTadini/ReactReview.git ````
* Instale as dependências: ```` npm install ````
* Inicie o servidor: ```` npm start ````
* Acesse o localhost no seu navegador.
